# 开发日志myOwnBlog项目 2026-04-10

## 今日完成内容

### 一、Blog 子页面结构搭建

- 新建 `pages/blog.html`
- 页面仅负责结构，不包含逻辑代码
- 采用模块化设计：

```text
Blog Page
├── 标题（page-header）
├── 分类栏（blog-filter）
└── 列表容器（blog-list）
```

核心思想：

> 页面只做“展示容器”，数据与逻辑全部交给 JS 处理

------

###  二、引入数据驱动（JSON）

新增数据文件：

```text
data/posts.json
```

用于存储博客信息：

- 标题（title）
- 描述（desc）
- 日期（date）
- 分类（tag）
- 内容路径（file）

实现效果：

- 页面不再写死内容
- 列表由 JS 动态生成
- 结构更清晰，易维护

------

### 三、Blog 列表渲染逻辑

在 `main.js` 中实现：

#### 1. 初始化函数

```js
initBlog()
```

作用：

- 请求 JSON 数据
- 调用渲染函数

------

#### 2. 渲染函数

```js
renderBlogList(posts)
```

功能：

- 遍历 JSON 数据
- 动态生成 `.blog-card`
- 绑定点击事件（openPost）

------

### 四、点击文章 → 打开详情页

实现函数：

```js
openPost(file, title, date)
```

核心功能：

- 使用 `history.pushState` 修改 URL
- 进入文章详情页

------

### 五、Markdown 内容系统

#### 1. 内容存储方式

使用：

```text
posts/*.md
```

优点：

- 写作体验好（Typora）
- 支持结构化内容（标题 / 列表 / 代码块）
- 易扩展

------

#### 2. 渲染方案

引入：

```html
marked.js
```

实现：

```js
marked.parse(md)
```

效果：

> Markdown → HTML 动态渲染
